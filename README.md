# Nothing — Tech Landing Page

Nothing is a sleek, minimalist landing page inspired by the London-based tech company. The project focuses on a clean "dot-matrix" aesthetic, featuring product showcases, category browsing, and a seamless mobile-first experience.

## Technologies Used

* **HTML5** — Semantic markup for structure and accessibility.
* **SCSS/SASS** — Advanced styling with mixins, variables, and nested rules.
* **BEM Methodology** — Consistent naming convention for scalable CSS.
* **Google Fonts** — Integration of 'Space Grotesk' and 'Space Mono' for a distinct tech feel.
* **JavaScript (ES6+)** — For handling the off-canvas menu and form interactions.

## Features

* **Mobile-First Approach** — Optimized for small screens first, ensuring a flawless experience on all devices using CSS Grid and Flexbox.
* **Unified Burger Menu** — A clean navigation strategy where the menu is always tucked away, keeping the focus on the content.
* **Product Catalog** — Beautifully structured sections for recommended products and category browsing with hover effects.
* **Interactive Contact Form** — A stylized feedback form with integrated validation and custom focus states.
* **Minimalist UI** — High-contrast design following the signature "Nothing" brand identity with "dot-matrix" typography.
